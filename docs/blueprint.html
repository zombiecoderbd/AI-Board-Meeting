🖥️
Frontend (Next.js Pages)
শুধু দেখানো + ব্যবহারকারীর input নেওয়া।
DB জানে না, সরাসরি query করে না।
API Route থেকে data নেয়, দেখায়।
v0.dev এখানে কাজ করবে।
🔀
API Routes (Next.js /api)
Frontend-এর request receive করে।
Settings পড়ে .env / config file থেকে।
Backend WebSocket-এ forward করে।
Edge TTS এখানে চলে।
⚙️
Backend (server/index.ts)
WebSocket server (port 3001)।
SQLite DB শুধু এখানে touch হয়।
AI provider (Ollama/Gemini) call করে।
Session + conversation সেভ করে।
Frontend → [HTTP/WS] → API Routes → [WS] → Backend → [DB + AI]

🧔
সলিমুদ্দিন
Sales & Marketing · বাজার বিশ্লেষক
SALES

পুরান ঢাকার ট্রেডার মেজাজে কথা বলে। বাংলাদেশের কাস্টমার সাইকোলজি, বাজারের হাওয়া, দাম-দর নিয়ে তুখোড়।

কথার স্টাইল: "ভাই, এইটা বেচতে গেলে আগে গল্পটা বেচেন।"

আপনি সলিমুদ্দিন — বাংলাদেশের দেশি বাজারের বিশেষজ্ঞ। পুরান ঢাকার ব্যবসায়িক মেজাজে কথা বলেন। বাংলাদেশের গ্রাহক মনোবিজ্ঞান, মূল্য নির্ধারণ, ডিজিটাল মার্কেটিং BD-context-এ ব্যাখ্যা করেন। মিক্স বাংলায় কথা বলুন, সরল ও বাস্তব পরামর্শ দিন।

ইঞ্জিনিয়ার বখতিয়ার
IT & Tech · সিস্টেম আর্কিটেক্ট
TECH

লজিক্যাল, দালালি সহ্য করেন না। লোকাল সার্ভার, নেট স্পিড, বাংলাদেশের infrastructure বাস্তবতা নিয়ে সৎ।

কথার স্টাইল: "ভাই, ক্লাউড দিয়ে স্বপ্ন দেখার আগে আপনার ISP ঠিক করুন।"

আপনি ইঞ্জিনিয়ার বখতিয়ার — BD tech reality-র সৎ মুখপাত্র। সিস্টেম আর্কিটেকচার, সিকিউরিটি, লোকাল ইনফ্রা নিয়ে বাংলাদেশের বাস্তবতায় কথা বলুন। দালালি বা overselling ধরিয়ে দিন। Technical terms বাংলায় ব্যাখ্যা করুন।

আপা
HR & People · টিম লিডার
HR

মায়া-মমতা দিয়ে কথা বলেন কিন্তু কাজের বেলায় কড়া। বাংলাদেশী অফিস কালচার, মানুষের মন বোঝেন।

API Contract — কে কোথায় request করবে

FE=Meeting Room | AD=Admin Panel | BOTH=দুইজনেই
🖥️ Meeting Room (Frontend) — এর Requests
MethodEndpointকাজকে
WS ws://localhost:3001 WebSocket connect, question পাঠানো, streaming response নেওয়া FE
GET /api/sessions Past meeting list (resume করতে) FE
GET /api/sessions/:id Specific session-এর conversation history FE
POST /api/tts Text পাঠাবে → MP3 আসবে (bn-BD-PradeepNeural) FE
GET /api/config Current model, provider, voice settings পড়বে BOTH
⚙️ Admin Panel — এর Requests
MethodEndpointকাজকে
GET /api/models/ollama Ollama locally installed model list AD
GET /api/models/gemini Gemini available model list (hardcoded) AD
PUT /api/config Default model/provider set করা AD
GET /api/agents সব agent persona list AD
PUT /api/agents/:id Agent system prompt / name / avatar edit AD
GET /api/sessions All past sessions list with stats AD
GET /api/agents/metrics Agent performance stats (best responses, score) AD
ℹ️
Config কোথায় থাকবে?
/api/config → server-side একটি JSON file বা SQLite-এর settings table (backend-এ) পড়বে। Frontend সরাসরি DB/file touch করবে না।

Bangla Voice — Edge TTS Strategy

msedge-tts · bn-BD-PradeepNeural · Free · No API Key
User কথা বলে / Type করে
→
Agent response আসে (WS)
→
Frontend /api/tts POST করে
→
Server msedge-tts MP3 বানায়
→
Browser Audio() play করে
→
Speaking indicator জ্বলে
// server/api/tts/route.ts import { MsEdgeTTS, OUTPUT_FORMAT } from 'msedge-tts'; const tts = new MsEdgeTTS(); await tts.setMetadata( 'bn-BD-PradeepNeural', // বাংলা পুরুষ ভয়েস OUTPUT_FORMAT.AUDIO_24KHZ_48KBITRATE_MONO_MP3 ); const mp3Buffer = await tts.toBuffer(text); // → Response.arrayBuffer() → browser play করবে
✅
কেন এটা: Microsoft Edge-এর internal TTS API। ফ্রি, rate limit নেই (personal use)। Neural voice — রোবটিক না। API key লাগে না।
